# 💳 SecureBank – Online Banking System

SecureBank is a modern, responsive online banking dashboard built using **HTML, CSS, and Vanilla JavaScript**. It provides users with essential banking features including **authentication**, **fund transfers**, **bill payments**, **account summaries**, and a **dynamic profile view** – all within a clean and sleek sidebar interface.

## 🚀 Features

- 🔐 **User Authentication**: Start with a secure login screen to authenticate users.
- 📊 **Dashboard Overview**: View a summary of your account details and balance.
- 💸 **Transfer Funds**: Securely transfer money between accounts.
- 🧾 **Pay Bills**: Instantly pay utility bills via a simple form.
- 👤 **User Profile**: View dynamically loaded user info (name, email, etc.).
- 📄 **Statements**: Access detailed transaction statements.
- 🚪 **Logout Option**: Cleanly exit the session via sidebar navigation.
- 🎨 **Modern UI**: Sidebar-based layout with a minimal flat design and color scheme.
